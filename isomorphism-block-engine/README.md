## 什么是跨域同构积木映射引擎？

跨域同构积木映射引擎（Isomorphism Block Engine）是 Y.MINE GameOS 体系下的**全域知识模组化拼装验证框架**。它依托**乐高四步拆解法**（拆零件→找接口→重新拼装→验证运行），将任意陌生领域拆解为标准化积木单元，通过接口兼容度计算完成跨领域结构同构映射，并由 **0.6 风险浓度熔断机制**做拼装合法性拦截。

**核心价值公式**：`元架构 + 积木映射 = 可工程化复用的全域同构拼装框架`

## 乐高四步拆解法（核心算法）

```
① 拆零件 → ② 找接口 → ③ 重新拼装 → ④ 验证运行
   ↓           ↓           ↓            ↓
模块化分层   接口同构    元架构重组    C≥0.6通过
```

- **拆零件**：将陌生领域（如CAD、电池、园林）拆解为独立的标准化积木单元
- **找接口**：识别积木之间的结构同构关系（如会计科目=图层=积木零件）
- **重新拼装**：按GameOS元架构规则重新组合积木，构建新领域的结构模型
- **验证运行**：通过兼容度C值计算，判定拼装链合法性（C≥0.6通过）

---

## 0.6 兼容度熔断机制

| C值区间 | 状态 | 处理 |
|---------|------|------|
| C ≥ 0.75 | 🟢 拼装通过 | 结构同构完成，可进入验证运行 |
| 0.6 ≤ C < 0.75 | 🟡 基本通过 | 可通过但建议优化接口匹配 |
| C < 0.6 | 🔴 熔断拦截 | 红色告警，禁止拼装通过 |

兼容度计算公式：
```
C = 接口匹配度 × 结构同构度 × 权重系数
```
> 精细权重系数、阈值参数位于私有引擎库 `isomorphism-block-private-engine/thresholds/`
