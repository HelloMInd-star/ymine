# 🔒 private_moodmind_engine · 金融私有闭源内核

**【核心禁区】**

本目录是 MoodMind-Lab 金融私有闭源内核占位目录。

**绝对规则：**
1. ❌ 本目录 **永远不提交任何代码** 到工程代码库
2. ❌ 工程层（public_dashboard/、public_api/ 等）**禁止依赖、禁止引用、禁止实现** 本目录下的任何内容
3. ❌ 所有金融底层数理公式、权重、周期逻辑、估值内核 **仅存在于本目录（实际部署时外置）**
4. ✅ 工程层仅通过 stub 层接收最终结果（11维向量、KMP分值、风险模长、状态标签）

**禁止在此实现的私有内核清单：**
- 11维向量底层计算公式
- I/P/D 三维PID自控迭代逻辑
- 四维光照动态行业权重算法
- 超球面风险模长求解公式
- 0.48/0.50/0.68三阈值底层判定逻辑
- 改良KMP周期匹配LPS底层源码
- 行情相位、箱体中枢、筹码结构私有算子
- 凯利仓位安全系数内核

**部署时：** 实际私有内核以独立加密包形式挂载，不通过工程代码库分发。
