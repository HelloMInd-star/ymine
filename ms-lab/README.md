## 🚀 快速启动

### 1. 环境要求
- Python 3.9+
- 浏览器（Chrome/Edge/Firefox 最新版）

### 2. 安装依赖
```bash
cd /path/to/ms-lab
pip install -r requirements.txt
```

`requirements.txt` 包含：
- `streamlit>=1.30`
- `plotly>=5.18`
- `pandas>=2.0`
- `cryptography>=41.0` （AES-256-GCM 加密所需）

### 3. 一键启动
```bash
bash scripts/start_dashboard.sh
```

启动成功后访问：
- 🌐 **入口页（静态）**：http://localhost:8090/ms-lab/index.html
- 📊 **Streamlit 仪表盘**：http://localhost:8501
- 🔙 **总控台入口**：http://localhost:8090/index.html

### 4. 手动启动
```bash
# 终端1：静态入口页
cd /path/to/ms-lab/..   # ms-lab 的父目录
python -m http.server 8090

# 终端2：Streamlit 仪表盘
cd mslab_dashboard
streamlit run app.py --server.port 8501 --theme.base dark
```

### 5. 环境变量
| 变量 | 默认值 | 说明 |
|------|--------|------|
| `MSLAB_PORT` | `8501` | Streamlit 端口 |
| `MSLAB_STATIC_PORT` | `8090` | 静态 HTTP 入口端口 |

## 🚫 强制红线（所有 Batch 严格遵守）

1. 禁止搭建通用应用题底层数学求解函数库（仅支持 2 套硬编码模板）
2. 禁止完整实现 128bit 高精度向量运算（仅预留空接口，调用即抛 `NotImplementedError`）
3. 禁止编写 KMP 底层匹配源码（只接收最终匹配分值）
4. 禁止开发算力调度底层博弈算法（仅做上报和指令接收外壳）
5. 禁止硬编码私有风控公式、金融估值深层算子
6. 程序不能自主生成新方程，仅可调用预设固定模板
7. 前端可视化只做展示，禁止嵌入底层数理运算逻辑

## 🔧 常见问题

**Q: Streamlit 启动端口被占用？**  
A: 启动脚本会自动清理旧进程；若需手动：`pkill -f "streamlit run.*app.py"`。

**Q: `ModuleNotFoundError: No module named 'cryptography'`？**  
A: 执行 `pip install cryptography` 或 `pip install -r requirements.txt`。

**Q: 128bit 密钥调用报错？**  
A: 按红线要求，128bit 仅预留参数位，调用 `get_key(128)` 会抛 `NotImplementedError`，等待 KMS 正式注入。

**Q: 批量导出为什么被拒绝？**  
A: 系统硬编码单次最多导出 `MAX_EXPORT_BATCH=100` 条，禁止全库批量导出。

**Q: 如何切换角色？**  
A: 侧边栏「🔑 权限角色（演示切换）」下拉框切换 operator/architect/admin，即时生效。
