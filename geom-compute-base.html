7.2 五类心智几何体・数理算力映射体系 V/S/L 三维算力模型

基础定义：

• 几何体总体积 V = 单次请求等效总 Token 消耗量

• 几何体外表面积 S = Prefill 阶段平方级算力开销

• 几何体轴向纵深 L = Decode 逐 Token 访存开销

有效算力浓度公式： η = (S·V) / (S·V + L·V2/3)

7.3 全套专业算力工程公式 Prefill/Decode/KV Cache 量化

① 等效总 Token 预估公式
Teff = α · V + β · S + γ · L
α/β/γ 为几何体-算力映射系数，V=体积，S=表面积，L=轴向纵深；综合估算单次推理等效 Token 消耗总量
② Prefill 计算密集型算力
Cprefill = S × dmodel2 × nlayer × FLOPsper_token
Prefill 阶段并行处理全部输入 Token，计算复杂度随序列长度呈平方级增长，对应几何体外表面积 S
③ Decode 访存密集型算力
Cdecode = L × (dmodel × nlayer × MKV + Mweight)
Decode 阶段逐 Token 自回归生成，访存带宽为瓶颈，计算量与生成长度 L 线性相关
④ 缓存复用算力节约率
Rsave = (1 − KVmiss / KVtotal) × 100%
公共前缀 KV 缓存命中比例越高，Prefill 阶段重复计算越少，算力节约率越高
⑤ KV 缓存显存占用公式
MKV = 2 × nlayer × nhead × dhead × Tctx × bbytes
KV Cache 显存占用 = 2(K/V) × 层数 × 头数 × 头维度 × 上下文长度 × 精度字节数（FP16=2, INT8=1）

⚡ 调度核心目标：在保障圆锥高价值用户实时独享算力的前提下，通过公共 KV 缓存复用 + 异步错峰队列 + 三层阈值熔断，实现全域算力资源成本最优、稳态可控。
