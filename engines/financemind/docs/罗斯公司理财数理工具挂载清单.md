## 第三章 力学同构映射规范（复用#30双层阻尼）

金融精算核心模型必须复用[exp30-gravity.html](file:///workspace/labs/structural-mechanics/exp30-gravity.html)双层阻尼模型映射逻辑：

| 力学概念（#30） | 财务概念（FinanceMind） | 映射关系 |
|----------------|----------------------|---------|
| C₁ 外层乱流阻尼 | **股权风险成本 Ke** | 市场波动、股权风险溢价 → 外层扰动 |
| C₂ 内层热风阻尼 | **债权刚性成本 Kd×(1-t)** | 利息支出刚性兑付、税盾缓冲 → 内层压力 |
| 球体总重量mg | **企业总价值 EV** | 总负载 |
| 偏心偏移量 cdist | **资本结构偏离度** | 偏离0.50中轴越远偏心越大 |
| 姿态稳定度 attStab | **财务健康度评分** | 0.49~0.51极致稳态对应最优杠杆 |
| 0.48安全线 | **保本线** | 对应保本不破产 |
| 0.68熔断线 | **破产风险熔断** | C₁+C₂总阻尼超过临界值，触发熔断降杠杆 |

## 第四章 阈值约束与风控规范

### 4.1 全局刚性阈值引用规范

```javascript
// 禁止本地重定义0.48/0.50/0.68数值，必须通过以下方式引用
const TH = (typeof THRESHOLDS !== 'undefined')
    ? THRESHOLDS
    : Object.freeze({ BREAKEVEN: 0.48, STEADY: 0.50, FUSE: 0.68 });
```

### 4.2 风控闸门触发条件

| 阈值 | 名称 | 触发动作 |
|-----|------|---------|
| **0.48** | 保本线（BREAKEVEN） | 触发保本预警，暂停高风险投资，启动现金流应急预案 |
| **0.50** | 稳态中轴线（STEADY） | 最优资本结构基准点，所有估值/博弈/资本配置以0.50为校准原点 |
| **0.68** | 熔断线（FUSE） | 触发全局熔断，自动降杠杆，阻断所有资本操作指令，经三角审计STEP9校验后方可恢复 |

### 4.3 资本结构最优杠杆锁定

- **最优资本结构中枢：** D/E = 1:1（对应资产负债率50%），即0.50稳态中轴线
- **安全区间：** 0.48 ~ 0.52（对应资产负债率48%~52%）
- **预警区间：** 0.45 ~ 0.48 或 0.52 ~ 0.68
- **熔断区间：** < 0.45 或 > 0.68
