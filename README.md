## 项目简介

**Y.Mine 不是又一个"知识管理工具"，也不是咨询公司的 PPT 模板。**

它是一套**演绎法驱动的精算操作系统**——将神经科学（行为决策代理）、金融精算（CAPM/凯利/夏普/杠杆）与前端工程（零依赖原生架构）熔于一炉，把"定价权"从经验直觉升级为可拆解、可对标、可模拟、可入库的数学推演系统。

底层用**行为神经科学**解释人类决策偏差，中层用**元架构**实现跨领域方法论迁移，顶层用**金融精算模型**输出可验证的定价与估值结论。所有模块通过 localStorage 数据总线实时串联，形成从信号采集到风控熔断的完整闭环。

**没有 npm install，没有 webpack，没有 node_modules。** 纯 HTML + CSS + 原生 JS。`git clone` → 双击 `index.html` → 开始精算。

### 💰 业财融合闭环（v1.6 核心突破）

**营销的"品牌认知/市场1-2-3维度"不再是漂浮的概念，而是可精算的财务成本。**

- **三流分发效率公式**：`分发效率 = 触达率 r₁ × 转化率 r₂ × 履约率 r₃`
- **机会成本抽象模型**：将营销资源占用映射为三维递减成本——资金时间价值(Rf)、边际递减(DIM)、最大回撤(MDD)
- **完整公式链**：总获客成本 TAC → 真实单位成本 → 边际贡献 → 营销ROI(ROMI) → 经营杠杆(DOL) → 真实杠杆ROI
- **三向数据流串联**：定价模型自动读取真实成本、6维风险向量OC滑块同步、CAPM风险溢价由ROMI动态修正
- **风控五档指示灯**：亏损/跑输无风险利率/健康/过热(>300%)/高效，实时告警

## 底层逻辑与数学模型

### 核心公式集

| # | 模型 | 公式 | 核心用途 |
|---|------|------|---------|
| 1 | **CAPM 资本资产定价** | $R = R_f + (R_m - R_f) \cdot \beta_{adj}$ | 必要收益率 = 无风险利率 + 风险溢价 × 杠杆调整β |
| 2 | **圆锥浓度模型** | $C = \frac{m \cdot \rho_0}{V(h)}$ | 定价权量化，0.48/0.50/0.68 三条稳态阈值 |
| 3 | **6维风险向量** | $\vec{v} = [f^*, L, SR, MDD, DIM, OC]$ | 凯利仓位/财务杠杆/夏普/最大回撤/边际/机会成本 |
| 4 | **向量模长** | $\|\vec{v}\| = \sqrt{\sum w_i v_i^2}$ | 综合风险强度度量 |
| 5 | **余弦相似度** | $\text{sim} = \cos(\theta) \cdot e^{-\|d\|}$ | 创始人画像 ↔ 案例向量匹配 |
| 6 | **总获客成本 TAC** | $TAC_{unit} = \sum C_i + \sum C_i r_i k_i$ | 营销三维度直接成本 + 机会成本 |
| 7 | **经营杠杆 DOL** | $DOL = \frac{P}{P - C_{real}} = \frac{1}{grossMargin}$ | 固定成本对利润的放大系数 |
| 8 | **营销投资回报 ROMI** | $ROMI = \frac{P - C_{real}}{TAC_{unit}}$ | 每1元营销投入带来的边际利润 |
| 9 | **真实杠杆 ROI** | $ROI_{real} = \frac{ROMI \cdot L}{1 + DOL \times 0.1}$ | 营销回报经财务杠杆放大后的真实收益率 |
| 10 | **凯利最优仓位** | $f^* = \frac{p(b+1)-1}{b}$ | 风险预算的最优资金分配比例 |

### 圆锥浓度 · 定价权三条铁律

```
0.68 ━━━━━━━ 稳态上限线（金融市场/AI旗舰稳态逼近，β=1.4-2.0）
         ▲ 高端定制 / AI 旗舰区间
0.50 ━━━━━━━ 定价权基准线（隐形冠军中位，β=1.0）
         ▲ B2B 隐形冠军区间
0.48 ━━━━━━━ 生死线（低于此线无定价权，β=0.7-0.9）
         ▲ 防御型基础 / 价格战区间
0.32 ━━━━━━━ 熔断线（瑞幸 9.9 价格战位置，β=1.8，f*=0.08）
```

**核心洞察**：浓度 $C$ 每超越 0.48 生死线 0.1，估值乘数提升 0.25x；但超过 0.68 稳态上限后，边际收益急剧递减，进入"过热区"，凯利仓位强制减半。

### 三步上手指南

```bash
# 1. 克隆仓库
git clone https://github.com/HelloMInd-star/ymine.git
cd ymine

# 2. 双击运行（不需要任何服务器）
open index.html            # macOS
# 或 xdg-open index.html   # Linux
# 或在文件管理器中双击 index.html

# 3.（可选）本地服务器（推荐用于跨页数据同步最佳体验）
python3 -m http.server 8000
# 然后浏览器访问 http://localhost:8000
```

### 一条完整精算链路

```
🔭 信息漏斗（7层筛选：熵→向量→信源→黑天鹅→宏观→红蓝海→国策）
       ↓ 信号提纯 100% → 20%
🧪 金融估值模拟器（6维向量 + 圆锥浓度 + CAPM + 业财融合TAC/ROMI/DOL）
       ↓ 精算输出
📡 AI定价基准库（算力$2.5/h · API $1/$3 · 订阅$20/月 · 分成25%）
       ↓ 行业对标
🗂️ 案例向量库（16案例：瑞幸+7隐形冠军+8AI，余弦相似度匹配）
       ↓ 画像匹配
```
