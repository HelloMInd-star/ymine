# Part 1 · 反向拆解器（以瑞幸为案例）

## 1.1 拆解方法论 · 六层反向追溯

反向拆解的核心命题：**从消费者看到的"终端零售价"出发，逐层向上回溯到"原料成本"，每一层都暴露一个定价决策点。**

```
┌─────────────────────────────────────────────────────────────┐
│  L6 · 终端零售价        9.9 元（消费者实付）                  │
│       ↑ 加价率 1.65x                                         │
│  L5 · 门店挂牌价        16.0 元（原价，用于锚定）              │
│       ↑ 加价率 1.23x                                         │
│  L4 · 渠道结算价        13.0 元（含平台佣金/配送费分摊）       │
│       ↑ 加价率 1.18x                                         │
│  L3 · 出库价            11.0 元（含门店租金/人力摊销）         │
│       ↑ 加价率 1.22x                                         │
│  L2 · 工厂出厂价         9.0 元（含生产/包装/损耗）           │
│       ↑ 加价率 1.50x                                         │
│  L1 · 原料成本           6.0 元（咖啡豆/奶/糖/杯材）          │
└─────────────────────────────────────────────────────────────┘
```

## 1.2 瑞幸六层价格层级拆解

| 层 | 名称 | 价格 | 加价率 | 决策点 |
|----|------|------|--------|--------|
| L6 | 终端零售价 | 9.9 元 | — | 价格战锚点，对标星巴克 30+ 元 |
| L5 | 门店挂牌价 | 16.0 元 | — | "原价"锚定，制造 38% 折扣幻觉 |
| L4 | 渠道结算价 | 13.0 元 | 1.23x | App 下单强制 → 私域流量沉淀 |
| L3 | 出库价 | 11.0 元 | 1.18x | 门店密度摊薄租金 |
| L2 | 工厂出厂价 | 9.0 元 | 1.22x | 自建工厂 + 规模采购 |
| L1 | 原料成本 | 6.0 元 | 1.50x | 全球直采 + 期货锁价 |

## 1.3 成本结构 · 三大成本池

### ☕ 直接成本池（变动成本）

| 项 | 单杯成本 | 占零售价 | 占直接成本 |
|----|---------|---------|-----------|
| 咖啡豆 | 1.2 元 | 12.1% | 20% |
| 奶 + 糖 | 1.5 元 | 15.2% | 25% |
| 杯材 + 盖 + 杯套 | 1.3 元 | 13.1% | 21.7% |
| 包装袋 + 配送耗材 | 0.8 元 | 8.1% | 13.3% |
| 损耗 + 报废 | 1.2 元 | 12.1% | 20% |
| **合计** | **6.0 元** | **60.6%** | **100%** |

### 🏪 间接成本池（半固定成本，按 200 杯/日分摊）

| 项 | 单杯分摊 | 占零售价 | 说明 |
|----|---------|---------|------|
| 门店租金 | 1.8 元 | 18.2% | 30-80 平米商圈店 |
| 人工（2-3 人/店） | 1.5 元 | 15.2% | 含社保 |
| 水电 + 网络 | 0.3 元 | 3.0% | — |
| 设备折旧 | 0.4 元 | 4.0% | 咖啡机 5 年摊销 |
| **合计** | **4.0 元** | **40.4%** | — |

### 📡 营销成本池（隐性补贴）

| 项 | 单杯分摊 | 占零售价 | 风险 |
|----|---------|---------|------|
| 新客补贴 | 1.5 元 | 15.2% | 老用户薅羊毛 |
| 优惠券折损 | 0.8 元 | 8.1% | 价格锚定伤害 |
| 品牌广告摊销 | 0.5 元 | 5.1% | 长期心智投资 |
| **合计** | **2.8 元** | **28.3%** | 营销费用率 > 毛利率 |

**总成本 = 6.0 + 4.0 + 2.8 = 12.8 元 vs 零售价 9.9 元 = 单杯亏损 2.9 元**

## 1.4 利润锚点 · 三条生死线

### 锚点 1 · 现金毛利线

```
现金毛利 = 零售价 - 直接成本 = 9.9 - 6.0 = 3.9 元（毛利率 39.4%）
```
**判断**：现金毛利为正，门店层面"看起来"赚钱，但忽略间接成本和营销。

### 锚点 2 · 门店净利线

```
门店净利 = 零售价 - 直接成本 - 间接成本 = 9.9 - 6.0 - 4.0 = -0.1 元（毛利率 -1%）
```
**判断**：单店层面已经不赚钱。规模扩张只是把"负毛利"放大 N 倍。

### 锚点 3 · 全链路净利线

```
全链路净利 = 零售价 - 全成本 = 9.9 - 12.8 = -2.9 元（毛利率 -29.3%）
```
**判断**：真正意义上的"卖一杯亏一杯"。靠资本输血和未来提价预期支撑。

## 1.5 可复用拆解框架 · 反向定价七步法

```
┌────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────┘
```

## 3.3 输出参数详解

### 输出 ① · 建议定价区间

```
输入示例：
  算力成本 = $2.5/h（H100 裸金属）
  目标客群 = 中型企业
  竞争对标 = 主流旗舰 (Gemini Pro $1.25/$5)
  生态位 = 主流跟随者

计算：
  成本价 = $2.5 × 65% 利用率 × 1/(200 tok/s × 3600s) = $0.022 / 1M tok
  客群溢价 = 2.0x（中型企业）
  竞品锚定 = 主流旗舰 $1.25/$5
  生态位目标毛利率 = 50%

  输入价 = max(成本 × 50x, 竞品 × 0.9) = max($1.1, $1.125) = $1.125
```
